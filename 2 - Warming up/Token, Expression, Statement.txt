1. Tokens
2+3; => 토근 4개
2 + 3 ; => 토큰 4개

1-1. 토큰의 종류
identifiers, Keywords, Punctuations, Operators, Literals

1-2. if
참과 거짓을 판단하기 위한 Keyword
i f => x

1-3. 공백
토큰을 구분
코드의 가독성
연산자와 피연산자 처리하는 방식 결정
tab, return 도 공백으로 취급

2. Expressions(표현식)
값, 변수, 연산자, 함수 등이 하나 이상 모여서 이루어짐.
표현식을 평가하다 : 하나의 결과값 도출하는 것
let x = 7
x를 평가하면 7이 된다
x + 1 => 3개의 표현식, 평가하면 8이 된다. (산술표현식)
x < 10  x가 7이면 참, 100이면 거짓 (논리표현식)

3. Statements(문장, 구문, 문)
대부분의 프로그래밍 언어에서는 문장 끝에 ; (세미콜론)으로 끝난다.
하지만 swift는 ;이 생략가능.

3-1. 문장의 종류
if, swift, guard, for-in, while
