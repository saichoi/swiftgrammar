Special Characters(특수문자)

! : 느낌표, 논리 부정, 옵셔널에서는 저장된 값 강제 추출
~ : Tilde
` : Back Tick
@ : At Symbol
# : Sharp
$ : Dollar Sign 파라미터 이름을 대체
% : percent sign 나머지 구할 때
^ : Caret 비트 연산
& : Ampersand 메모리 주소 얻을 때, 참조 전달
* : Asterisk 곱하기 연산
() : Parentheses 함수 호출, 계산 순서 지정
- : Minus Sign/Hyphen 빼기 연산
_ : underscore 와일드 카드 패턴에서 사용됨
= : Equal Sign 변수, 상수에 값을 저장
+ : Plus Sign 더하기 연산
[] : Square Bracket 컬렉션에 저장된 값에 접근할 때 사용
{} : Brace 코드 블럭의 범위를 지정할 때
\ : Backslash 스트링인터페이스, 키프레스 문법에서 사용
| : Piple/Vertical Bar 논리연산, 비트연산
; : Semicolon 문장 마지막
: : Colon 자료형 지정, 자료형에서 키와 값을 구분 
, : Comma 함수 혹은 배열 전달 값 나열, 
. : Period 매서드 호출, 속성 접근
<>: Angle Bracket 크기 비교, 제네릭 형식 파라미터 지정
/ : Slash 경로지정
? : Question Mark Optional에서 주로 사용
